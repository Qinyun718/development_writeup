.01 Goal & Value

The application presents multiple heroes in the game of “Glory of Legend” with their basic information, including name, gender, role, difficulty and price. It provides users with the general information to choose the target hero they want to play. The shopping cart offers the convenience to add and remove the selected heroes after they evaluate the total price.

Usability Principles

User Control and Freedom: Users often perform actions by mistake. If the user clicks the filtering or sorting function by mistake, this application provides “All” and “Default” button to allow users to go back to the original setting.

Consistency: The application provides the consistency when displaying the items, no matter in general view or in the view of the shopping cart.

Minimalist design: The application provides the easiest way for the users to select their target heroes with the most straightforward filtering and sorting functions. Users could understand the purpose of the application when taking an easy look to the website.
